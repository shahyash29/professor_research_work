Circle on the Map

Display a map highlighting a circular region

This example shows a circle with a radius of 50000 meters centered over a location in Los Angeles (34.019929°N, -118.503893°E) and displayed on a moveable map.

Code

A circle is created using the H.map.Circle class, passing in the center point and radius. The look-and-feel of the circle can be altered by adding the style parameter.
